I worked in Dauntless for around 1 year, focusing on UI engineering of existing and new features. I was part of a team called "Systems" that was responsible for core UI systems, features in Menus and gameplay

Feature - Ramsgate map

Tech Stack

  • Unreal Engine 4
  • UMG
  • C++

One of the main features that I owned was the Ramsgate map, which was a realtime map of the main city hub in Dauntless. I was responsible of creating the widget from scratch, load the data using Data Tables and implement all the interactions using UMG and C++.

Also I was able to extend the Engine code to support this kind of Camera View position to 2D coordinates conversion, allowing to easily position elements on the map based on world coordinates.
