I worked on Dauntless for approximately 1 year, focusing on UI engineering for existing and new features. I was part of a team called "Systems" that was responsible for core UI systems, menu features, and gameplay.

Feature - Ramsgate Map

Tech Stack

  • Unreal Engine 4
  • UMG
  • C++

One of the main features I owned was the Ramsgate map, which was a real-time map of the main city hub in Dauntless. I was responsible for creating the widget from scratch, loading the data using Data Tables, and implementing all interactions using UMG and C++.

I was also able to extend the Engine code to support Camera View position to 2D coordinate conversion, allowing easy positioning of elements on the map based on world coordinates.
