Feature - Music Library - Fortnite Festival

Tech Stack

  • Unreal Engine 5
  • UMG
  • Common UI
  • C++

During my time working on Fortnite, I contributed to the Music Library feature that was released for the Fortnite Festival game mode, performing bug fixes related to navigation, asset loading, and gamepad support.

The Music Library is a feature that allows players to select songs to be played during the game, choose difficulties, manage their playlists, and browse leaderboards.

Feature - Message of the Day

Tech Stack

  • Unreal Engine 5
  • UMG
  • Common UI
  • C++

I was one of the main developers for the MOTD system in Fortnite. The MOTD is a Common Activatable Widget that displays dynamic content. I was responsible for updating the main widget and components using UMG, Blueprints, and C++.

The dynamic content was created using a custom CMS that Epic Games developed for Fortnite. The content was fetched using a REST API, then parsed from formatted JSON and displayed on the MOTD widget. It supports different types of content such as images, videos, and customizable buttons.

Each button implementation needed to be flexible enough to support different actions, such as opening a URL, navigating to a different part of the UI, or launching a specific game mode.
