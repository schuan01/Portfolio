The main objective was to teach the team how to use Unreal Engine 5 for UI implementation, from the most basic widgets to complex screens.

Feature - Common UI Migration

Tech Stack

  • Unreal Engine 5
  • UMG

The main challenge in this project was to migrate the existing UI codebase to use the Epic Games Common UI framework. This involved refactoring existing widgets, Blueprints, and C++ code to be compatible with Common UI standards and practices.
